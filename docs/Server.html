Class: Server

Server

new Server(ip, port, timeout)

Creates a server instance. The server is used to accept multiple incoming connections and to send data to those connections
Parameters:
Name Type Description
ip string ip ip address of server
port int port to connect to
timeout int The number of milliseconds of inactivity before a socket is presumed to have timed out. 0 will disable the timeout behavior
Source:
  • PubSub/server.js, line 14
Example
var server = new Server('127.0.0.1', 1337, 0);
